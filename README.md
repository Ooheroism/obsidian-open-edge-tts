# Read in Edge

Open the active Markdown file in Microsoft Edge with its built-in **Read Aloud** text-to-speech feature.

[中文](README.zh-CN.md)

## Features

- Ribbon icon to open the current file in Edge
- Workspace command: *Read with Edge*
- Desktop only

## Usage

1. Open a Markdown note in Obsidian.
2. Click the ribbon icon or run `Read with Edge` from the command palette.
3. In the Edge window that opens, use `Ctrl + Shift + Right-click` → **Read aloud** to start TTS.

## Install

### From source

Put the compiled `main.js` and `manifest.json` into your vault’s plugins folder and enable it in Settings → Community plugins.

### From GitHub Releases

Download `main.js` and `manifest.json` from the latest release and put them into your vault’s plugins folder:

```
<vault>/.obsidian/plugins/read-with-edge/
```
